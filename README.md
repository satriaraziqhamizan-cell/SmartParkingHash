# SmartParkingHash
Smart Parking System With Hash Function adalah sistem parkir cerdas yang menggunakan fungsi hash dari 4 digit terakhir plat kendaraan untuk menentukan slot parkir otomatis. Sistem mendukung kategori motor, mobil, dan bus, serta menangani collision dengan linear probing. Menampilkan status slot secara real-time dan lokasi parkir otomatis.
